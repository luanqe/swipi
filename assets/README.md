# Assets Needed

Please add the following image files to this directory:

## Required Files

1. **icon.png** (1024x1024)
   - App icon for iOS/Android
   - Should contain the Swipi logo
   - Format: PNG with transparency
   - Colors: Primary Blue (#0A66C2)

2. **splash.png** (1242x2436 for iOS)
   - Launch screen image
   - Same design as SplashScreen component
   - Format: PNG
   - Background: Gradient (#0A66C2 to #004182)

3. **adaptive-icon.png** (1024x1024)
   - Android adaptive icon
   - Foreground layer only
   - Background color in app.json

4. **favicon.png** (48x48)
   - Web favicon
   - Format: PNG

## Temporary Solution

The app will run without these files but you'll see warnings. The custom SplashScreen component will be used instead.

## Design Guidelines

- Use the Swipi card logo (two overlapping cards)
- Primary color: #0A66C2 (LinkedIn Blue)
- Professional, clean design
- High contrast for visibility
